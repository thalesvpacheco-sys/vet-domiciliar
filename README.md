# Dra. Rilquia Horrana — Site Institucional

Landing page profissional para a **Dra. Rilquia Horrana**, médica veterinária especializada em atendimento domiciliar em Goiânia/GO.

## Sobre o projeto

Site institucional desenvolvido pela [Grifo Agency](https://grifo.agency) com foco em:

- Apresentação dos serviços de atendimento veterinário domiciliar
- Vitrine de produtos veterinários
- Página dedicada de Perícia Forense Veterinária
- Captação de leads via WhatsApp

## Stack

- HTML5, CSS3 (Tailwind via CDN)
- Vanilla JavaScript (ES2020+)
- Hospedagem: GitHub Pages

## Estrutura

```
vet-domiciliar/
├── index.html              # Landing page principal
├── forense.html            # Página de perícia forense
├── assets/
│   ├── css/                # Estilos customizados
│   ├── js/                 # Scripts modulares
│   └── img/                # Imagens organizadas por contexto
└── data/
    └── produtos.json       # Catálogo de produtos
```
